First JS Excersize

This excersize converts pounds to kilograms.

We take in a number in pounds(lbs) and convert it to kilograms(kg).

This excersize includes

  • Linking this page to the *.js file.
  • Using Javascript to accept a number input in pounds(lbs).
  • Converting the number to kilograms (* 0.454) in js.
  • Reporting the weight in kg to the screen.
  • Logging the message "Conversion Complete" to the console.
